EcmaScript6 Examples - rest parameters

EcmaScript 6 brings a different way to pass parameters to a function. It is called rest parameters and let the function receives a variable number of values as an array.

Let's see an example: 

                
                    var sum = function(...numbers){
                        let result = 0;

                        numbers.forEach(function(_number){
                            result += _number;
                        });

                        console.log(result);
                    };

                    sum();
                    sum(4,5);
                    sum(4,5,1,7,4);

The function receives a variable number of values, and work with this parameter as an array. If the function doesn't receive any value it is an empty array. 

                
                    var sum_accum = function(result, ...numbers){
    
                        numbers.forEach(function(_number){
                            result += _number;
                        });

                        console.log(result);
                    };

                    sum_accum(0);
                    sum_accum(1, 4,5);
                    sum_accum(0, 4,5,1,7,4);

The rest parameter can be together with other variables in a function, but it have to be the last parameter in the function because it receives a variable number of values.
