EcmaScript6 Examples - spread

Templates string in EcmaScript 6 is a form to build strings filling gaps with variables without use concatenation. Templates string use the expression with ${expression} to fill the gap with a variable value.

Let's see an example: 

                
                    var template = function(x, y, z){
                        let color = "white";
                        let size = "big";

                        console.log("The cat is ${color} and ${size}.");
                        console.log(`The cat is ${color} and ${size}.`);
                    };

                    template();

Template strings doesn't work in a string with double quotes. To the string works with a template strings expression is necessary that it is surrounded by backquotes.
