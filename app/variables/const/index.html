EcmaScript6 Examples - const

The const keyword came to EcmaScript6 to allow the language works with constants and have immutable values. Const keyword will create a immutable variable which can't be reassigned after to be created.

Let's see an example: 

                
                    var const_es5 = function(){
                        var a = 5;

                        if(a === 5){
                            a = 6;
                            console.log(a);
                        }


                        console.log(a);
                    }

                    const_es5();

In EcmaScript5 didn't exist anything to work with constants, the most closest of this could be use var and to pay attention to not reassigned the variable. 

                
                    var const_es6 = function(){
                        const a = 5;

                        if(a === 5){
                            //a = 6;
                            console.log(a);
                        }


                        console.log(a);
                    }

                    const_es6();

In EcmaScript6 is possible to declare a constant, but is necessary to pay attention to not change its value because it will cause a error message.
