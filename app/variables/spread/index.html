EcmaScript6 Examples - spread

EcmaScript 6 brings a different way to pass values to parameters a function or fill an array with values. The name of this is spread, and it is one way to pass an array to n variables.

Let's see an example: 

                
                    var spread_func = function(x, y, z){
                        console.log(x + y + z);
                    };

                    spread_func(...[1,2,3]);

The function receives an array with three values which will populate the three variable of function. 

                
                    var spread = function(){
                        let arr = [3,4];
                        let arr2 = [1,2,...arr,5,6,7];
                        
                        arr2.forEach(function(_number){
                            console.log(_number);
                        });
                    };

                    spread();

The array is populated with another array in middle of ordinary values.
