EcmaScript6 Examples - array - find

EcmaScript 6 has a new array method which makes easy to looking for a value inside an array.

Let's see an example: 

                
                    let arr1 = [1,3,5,10,12];
                    let match = arr1.find(n => n > 5);
                    console.log(match); //=> 10

                    match = arr1.find(n => n > 50);
                    console.log(match); //=> undefined

                    let find_bigger_than_five = (element, index, array) => {
                        console.log(element); //=> 1, 3, 5, 10
                        console.log(index); //=> 0, 1, 2, 3
                        console.log(array); //=> [1,3,5,10,12]
                        if(element > 5){
                            return element;
                        }
                    };
                    match = arr1.find(find_bigger_than_five);
                    console.log(match); //=> 10

Find method from an array can receive a arrow function or a common function. In both cases the find method will looking for the value according the statement informed, and it will always return one value or undefined. 

                
                    match = arr1.findIndex(n => n > 5);
                    console.log(match); //=> 3

                    match = arr1.findIndex(n => n > 50);
                    console.log(match); //=> -1

                    match = arr1.findIndex(find_bigger_than_five);
                    console.log(match); //=> 3

The findIndex method from an array is similar of find method, but instead to return the value it returns the index. Another difference is that if the statement doesn't find nothing it returns -1 instead the undefined value.
