EcmaScript6 Examples - array - of

EcmaScript 6 has a new method in Array object which transform an array-like or iterable in a truth array.

Let's see an example: 

                
                    let arrLike = window.document.querySelectorAll('div');
                    console.log(arrLike);
                    console.log(arrLike.forEach);

                    let fromArr = Array.from(arrLike);
                    console.log(fromArr);
                    console.log(fromArr.forEach);

When used the method querySelectorAll to grab something from UI it will return an array-like. For example, if is necessary to have a loop among the return of the UI elements is necessary has a truth array, and to do that is necessary to convert the array-like in a truth array using the Array.from.
