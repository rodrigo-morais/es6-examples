EcmaScript6 Examples - inheritance

Another feature which came with classes in EcmaScript 6 was inheritance. Now is possible that one class to inherit of another.

Let's see an example: 

                
                    class Person{
                        constructor(name){
                            this._name = name;
                        }

                        get name(){
                            return this._name;
                        }

                        set name(value){
                            this._name = value;
                        }
                    }

                    class Employee exyends Person{
                        doWork(){
                            return `${this._name} is working.`
                        }
                    }

                    let employee = new Employee('Rodrigo');

                    employee.name; //=> Rodrigo
                    employee.doWork(); //=> Rodrigo is working.

To a class to inherit from another is necessary to use the keyword "extends" and the class will inherit all attributes, properties, and methods from another class.
