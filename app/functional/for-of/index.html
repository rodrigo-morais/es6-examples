EcmaScript6 Examples - for of

EcmaScript 6 brings a new way to iterate using the keyword "for", now is possible to iterate a iterable element with "for of" which gets the value of object. When an element becomes an interable element it will be an array with this format: [ { value: , done: false }, { value: , done: false }, { value: , done: false } ] This array has a method called "next" which allows navigate among each item. When is used the "for in" it takes the index of element, and "for of" takes the value.

Let's see an example: 

                
                    let numbers = [6,9,1,3,8];
                    let iterator = numbers[Symbol.iterator]();
                    let next = iterator.next();

                    while(!next.done){
                        console.log(next); //=> {value: 6, done: false} - {value: 9, done: false} - ...
                        next = iterator.next();
                    }


                    for(let n in numbers){
                        console.log(n); //=> 0 - 1 - 2 - 3 - ...
                    }

                    for(let n of numbers){
                        console.log(n); //=> 6 - 9 - 1 - ...
                    }

When the array is converted in a iterable object is possible to take the values. Using "for in" it will catch the indexes and not the values, on the other hand the "for of" will catch the values.
