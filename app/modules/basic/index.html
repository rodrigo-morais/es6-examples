EcmaScript6 Examples - modules - basic

Until ES5 didn't exist any way to work with modules nativily, to do that was necessary to use CommonJS or AMD. ES6 is bringing the modules to the core of language using the similar issues from CommonJS and AMD.

Let's see an example: 

                
                    //.employee.js
                    export class Employee {
                        constructor(name) {
                            this._name = name;
                        }

                        get name() {
                            return this._name;
                        }

                        doWork() {
                            return '${this._name} is working';
                        }
                    }

                    //.module.js
                    import Employee from './employee.js'

                    var employee = new Employee('Rodrigo');
                    employee.doWork(); //Rodrigo is working

The component, class, or piece of code which will be imported by another file has to use the reserved word "export" to becomes exportable. To import a file is necessary to use the reserved word "import", and then the file is imported in a variable.
