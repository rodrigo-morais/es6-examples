EcmaScript6 Examples - modules - multiples

ES6 brings a new feature which allows that a same file to export more than one module.

Let's see an example: 

                
                    //.exports.js
                    export class Employee {
                        constructor(name) {
                            this._name = name;
                        }

                        get name() {
                            return this._name;
                        }

                        doWork() {
                            return '${this._name} is working';
                        }
                    }

                    export let emp = new Employee('Morais');

                    export let number = 12;

                    //.imports.js
                    import {Employee, emp, number} from './exports.js';

                    var employee = new Employee('Rodrigo');
                    employee.doWork(); //Rodrigo is working

                    emp.doWork();

                    console.log(number);

Is possible that one file to export more than one element (class, object, native object, and etc). To import it is necessary to receive each object with correct name and order.

Is possible that the import file uses a variable to receive all element which are exported. In this case this variable is an object which has inside of itself all element exported.
